## Rodando localmente

Clone o projeto

```bash
  git clone git@github.com:ed-cavalcanti/education-landing-page.git
```

Entre no diretório do projeto

```bash
  cd education-lainding-page
```

Abra o arquivo index.thml
Ou
Utilize a extensão Live Server no VSCode
